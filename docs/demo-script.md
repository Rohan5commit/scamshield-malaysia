# Demo Script

## Narrative

ScamShield Malaysia is positioned as a trust-and-safety layer for scam prevention, fraud awareness, and digital literacy in Malaysia.

## Flow

1. Open the landing page and frame the Malaysian scam problem.
2. Paste a suspicious message and show the structured risk score.
3. Switch to URL mode and show phishing-specific red flags.
4. Upload a screenshot and explain mock mode versus live Gemini mode.
5. Submit a sanitized community report and search the feed.
6. Close on scalability: banks, telcos, public agencies, and Cloud Run deployment readiness.
